# MK-VS360

MK-VS360 is a single-page 360 degree virtual showroom workflow tool. It starts from a 2:1 equirectangular panorama, converts it into six editable cube faces, lets you re-upload the edited faces, merges them back into a 360 panorama, and exports a shareable virtual tour.

Main tool file:

```text
index_cloud_hq.html
```

## Features

- Upload a 2:1 equirectangular 360 panorama.
- Split the panorama into six cube faces.
- Download the cube faces as a ZIP for external editing.
- Re-upload edited cube faces.
- Merge faces into a 3x2 cube map.
- Convert the cube map back to a 2:1 panorama.
- Preview the final scene in a 360 viewer.
- Add info and teleport hotspots.
- Export a standalone 360 web package.
- Export a GitHub Pages ZIP for free high-quality sharing.

## Basic Workflow

Open the tool in a browser:

```text
index_cloud_hq.html
```

Recommended workflow:

1. Upload a 2:1 equirectangular panorama.
2. Split it into six cube faces.
3. Edit the faces in external design or retouching software.
4. Re-upload the edited faces.
5. Merge and convert back to a 2:1 panorama.
6. Preview the 360 tour in Step06.
7. Add hotspots if needed.
8. Export the 360° Web Package or GitHub Pages ZIP.
